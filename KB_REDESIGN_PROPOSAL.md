# 🎨 Предложение по редизайну Базы Знаний

## 📊 Анализ топовых KB систем

### 1. **Notion** (notion.so)
✅ Боковая панель с древовидной навигацией
✅ Страницы могут быть вложены на любую глубину
✅ Быстрый поиск (Cmd+K)
✅ Иконки для визуальной идентификации
✅ Недавние страницы, Избранное

### 2. **Confluence** (Atlassian)
✅ Пространства → Страницы → Подстраницы
✅ Дерево страниц слева
✅ Хлебные крошки сверху
✅ Метки/теги для гибкой организации
✅ Table of Contents справа

### 3. **GitBook**
✅ Книжная структура: Главы → Страницы
✅ Поиск с живыми результатами
✅ Навигация previous/next
✅ Версионирование контента

### 4. **Obsidian**
✅ Граф связей между статьями
✅ Внутренние ссылки [[Article Name]]
✅ Теги для группировки
✅ Backlinks (кто ссылается на эту статью)

## 🚀 Рекомендация: **Гибридный подход**

Комбинация лучших практик:

### Структура:

1. **Боковая панель (Sidebar):**
   - 📚 Главная
   - ⭐ Избранное
   - 🕐 Недавние (последние 10)
   - 📊 Популярные
   - ─────────────
   - 📁 Все статьи (древовидный список)
   - 🏷️ По тегам (облако тегов)
   - ─────────────
   - [+ Создать статью]

2. **Главный контент:**
   - Хлебные крошки сверху
   - Статья по центру
   - Table of Contents справа (для длинных статей)

3. **Поиск:**
   - Глобальный поиск вверху (Cmd+K)
   - Живые результаты с preview
   - Фильтры: по тегам, автору, дате

4. **Организация:**
   - **Основная:** Теги (гибкие, множественные)
   - **Дополнительная:** Категории (опционально, для структуры)
   - **Автоматическая:** Недавние, Популярные, Избранное

## 🎨 UI/UX улучшения

### 1. **Командная палитра (Cmd+K)**
```
┌─────────────────────────────────────┐
│ 🔍 Поиск статей, команд...         │
├─────────────────────────────────────┤
│ 📘 Настройка системы               │
│    Инструкции > Настройка           │
│ 📗 Добавление сотрудников          │
│    Инструкции > Сотрудники          │
│ ─────────────────────────────────  │
│ ⚡ Создать статью                   │
│ ⚡ Управление категориями (admin)   │
└─────────────────────────────────────┘
```

### 2. **Быстрые действия**
- Кнопка "Редактировать" появляется при наведении (для авторов)
- Кнопка ⭐ для быстрого добавления в избранное
- Кнопка 🔗 для копирования ссылки
- Кнопка 📤 для экспорта (PDF, Markdown)

### 3. **Связи между статьями**
```markdown
Смотрите также: [[Добавление сотрудников]]

Для настройки прав см. [[Управление доступом]]
```
→ Автоматически создаёт ссылки и показывает backlinks

### 4. **Граф связей** (для админа)
Визуальная карта связей между статьями
